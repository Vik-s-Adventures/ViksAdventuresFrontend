FRACCIONES

INTRODUCCIÓN A LAS FRACCIONES

  • Las fracciones son útiles para expresar partes de un todo o dividir algo en partes iguales.

PARTE DE UN TODO

  • Una fracción representa una división en partes iguales.

FRACCIÓN COMPLEMENTARIA

  • Es la parte restante del todo cuando se ha utilizado una porción.
  • Ejemplo: Si usas 2/3, la fracción complementaria es 1/3.

HOMOGENEIZAR

  • Consiste en uniformizar los denominadores de fracciones con denominadores diferentes para poder operarlas.

FRACCIONES EQUIVALENTES

  • Representan la misma cantidad aunque tengan numeradores y denominadores diferentes.

SUMA DE FRACCIONES

  • Alinea las fracciones con el mismo denominador para sumarlas.

RESTA DE FRACCIONES

  • Resta las fracciones con el mismo denominador directamente.

DIVISIÓN DE FRACCIONES

  • Multiplica en cruz: numerador por denominador y viceversa.

MULTIPLICACIÓN DE FRACCIONES

  • Multiplica los numeradores entre sí y los denominadores entre sí.

CONSEJOS

  • Dos fracciones son equivalentes si representan la misma parte del todo.
  • Para sumar o restar fracciones con el mismo denominador, basta con sumar o restar los numeradores.
  • Homogeneiza las fracciones para hacerlas comparables si tienen denominadores diferentes.
